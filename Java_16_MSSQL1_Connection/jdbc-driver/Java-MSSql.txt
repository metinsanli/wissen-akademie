ECLIPSE ICIN:
1. Microsoft JDBC Driver for SQL Server indirmek için şu adrese gidilir : http://msdn.microsoft.com/en-us/sqlserver/aa937724.aspx
2. sqljdbc_4.0.2206.100_enu.tar.gz isimli dosya indirilir. (dosya adı versiyona göre değişebilir)
3. sqljdbc4.jar dosyası Eclipse'de proje klasörüne kopyalanır.
4. Eclipse'te proje klasörüne sağ tıklanır ve "Properties" seçilir.
5. Açılan pencereden "Java Build Path / Libraries / Add External Jars..." açılıp daha önce proje klasörüne kopyaladığımız "sqljdbc4.jar" isimli dosya seçilir.

MSSQL SERVER ICIN :
1. "Control Panel \ All Control Panel Items \ Windows Firewall \ Advanced Settings" açılır.
2. "Windows Firewall with Advanced Security on Locl Computer \ Inbound Rules" a sağ tıklayıp "New Rule..." seçilir ve 1433 nolu port haberleşmesi açılacak şekilde ayarlamalar yapılır.
3. Aynı port aynı seçenekler ile "Outbound Rules" kısmına da tanımlanır.
4. "Start Menu \ All Programs \ Microsoft SQL Server 2008 \ Configuration Tools \ SQL Server Configuration Manager" açılır.
5. "SQL Server Network Configuration \ Protocols for MSSQLSERVER" seçilir "TCP/IP" "Enabled" olarak değiştirilir.
6. Bu aşamadan sonra SQL SERVER Servisini yeniden başlatmanız istenecektir.
7. "My Computer" simgesine sağtıklanır "Manage" seçilir. "Server Manager(Server İsmi) \ Configuration \ Services" seçilir.
8. Servisler arasından "SQL Server (MSSQLSERVER)" servisi bulunur ve sağ tıklanır. "Restart" seçilir.

Bitti. Java kaynak kodunuzu çalıştırıp SQL bağlantısını deneyebilirsiniz.
